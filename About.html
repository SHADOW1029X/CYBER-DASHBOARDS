The Six Pages

Login System
Auth Gateway

The authentication entry point. WebGL background, credential validation, Cloudflare D1 session management, and AbortController-guarded fetch with configurable timeout fallback.

Secure Auth Flow
PBKDF2 password hashing, constant-time comparison, and HttpOnly cookie session tokens with 7-day TTL rotation.
Edge Validation
All validation runs at the Cloudflare Worker layer. SQL injection prevention, rate-limiting, and D1 parameterized queries by default.
Timeout Guards
AbortController signals cancel stalled requests after 8s. Visual countdown feedback and graceful error state recovery built in.
Auth Sequence
1
User submits credentials — client-side format validation fires before any network request.
2
AbortController-wrapped fetch hits the Cloudflare Worker within 8s timeout.
3
Worker queries D1 with parameterized SQL, verifies hash, issues session cookie.
4
Client stores session reference in memory, redirects to Dashboard on 200 OK.
Cloudflare WorkersD1 SQLAbortControllerPBKDF2HttpOnly Cookies
Registration System
User Onboarding

Multi-step account creation with real-time field validation, username uniqueness checks against D1, and role assignment. Progressive disclosure reduces cognitive load across three form stages.

Progressive Steps
Three-stage form with animated transitions. Each step validates independently before unlocking the next, reducing form abandonment.
Live Validation
Username uniqueness verified on blur via debounced D1 lookup. Password strength meter updates in real time with entropy scoring.
Security Headers Suite
CSP with default-src 'none', X-Content-Type-Options: nosniff, X-Frame-Options: DENY, Referrer-Policy: no-referrer enforced on all registration endpoints.
Multi-step FormsD1 Uniqueness CheckEntropy ScoringCSP Headers
Analytics Dashboard
Data Command Center

Live metrics, Canvas-rendered charts, and real-time D1 queries. Built for decision-makers who need sub-second data refresh without a full page reload.

Canvas Charts
Line, bar, and scatter plots rendered directly on Canvas 2D. No chart library — pure geometry from raw D1 query results, animated on data update.
Live Metrics Stream
Polling loop with configurable interval. Differential updates applied to DOM — only changed nodes re-render, keeping the interface responsive.
Export Pipeline
One-click CSV, TXT, and PDF export via Blob API. No server round-trip — data serializes client-side and triggers browser download directly.
Canvas APIBlob APIFetch APID1 SQLCSV Export
User Profile
Identity Management

Editable user profile with avatar upload, bio editing, and settings persistence. Changes optimistically update the UI before the D1 write confirms.

Optimistic Updates
UI state changes instantly on user action. Background D1 write confirms asynchronously. Rollback triggers on error with user notification.
Avatar Processing
Client-side canvas resize and compression before upload. MIME type validation, 2MB cap, and WebP conversion for bandwidth efficiency.
Preference Sync
Theme, notification, and accessibility settings stored in D1 user record. Applied on session load so preferences follow the user across devices.
Canvas ResizeWebP ConversionOptimistic UID1 Preferences
Leaderboard
Live Rankings

Ranked user table with live D1 queries, animated rank transitions, and search/filter. Virtualized rendering handles thousands of rows without scroll jank.

Virtual Scroll
DOM recycling renders only visible rows. 10,000-entry boards scroll at 60fps with no layout thrash or memory growth.
Rank Animations
FLIP-technique animations fire when ranks shift. Entries slide to new positions with spring physics — no layout recalculation mid-animation.
Search Index
Client-side trie search across usernames. Results appear on every keystroke with zero network requests using pre-fetched session data.
Virtual DOMFLIP AnimationTrie SearchD1 Rankings
